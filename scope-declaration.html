1. Project Identity & Purpose

QUAZENTHOR (QWBQ) is a standard, fixed-supply BEP-20 token deployed on BNB Smart Chain mainnet. It serves as a public educational reference implementation that enables real interaction with core BEP-20 functions: transfers, approvals, allowance checks, event logging, and holder distribution observation.

The token is designed for students, developers, and researchers who want to study immutable smart contract behavior in a live blockchain environment ‚Äî including real gas usage, transaction finality, and event emission.

The name and symbol are used only for project identification and have no affiliation with any existing entity or brand.

QWBQ enables learners to:

  • Test approve / allowance mechanics
  • Monitor Transfer and Approval events
  • Observe changes in holder distribution over time
  • Verify fixed supply behavior on mainnet

All interactions consume real gas and are permanently recorded on the BNB Smart Chain.
