1. Introduction

This privacy policy outlines how the QUAZENTHOR educational reference project handles information. As a static informational website, we prioritize transparency and minimal data practices.

QUAZENTHOR (QWBQ) is a fixed-supply BEP-20 token on BNB Smart Chain mainnet, designed for studying standard token mechanics in a live environment.

2. Data Collection

This website does not collect, store, or process any personal information from visitors.

No user accounts, forms, logins, or interactive features exist that could gather data.

3. Cookies & Tracking

No cookies are used. No tracking scripts, analytics tools, or third-party embeds are present.

The site is fully static HTML/CSS with no JavaScript required for core access.

4. Blockchain Interactions

Token interactions occur directly on BNB Smart Chain via users' own wallets. This website does not handle or access any blockchain data.

Links to public explorers (e.g., BscScan) are provided for verification, but no personal data is involved.

5. Server Logs

Standard web server logs may temporarily record access information (e.g., IP addresses) for operational purposes only. These are not used for profiling or stored long-term.

6. Third Parties

No data is shared with third parties. The site has no integrations with external services that could track users.

7. Changes to Policy

Any updates to this policy will be documented in the changelog and reflected on this page.

8. Contact
